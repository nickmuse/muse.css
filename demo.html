muse.css

What

🔑︎ This is my re-usable CSS file

Why

Because I don't want to write any more CSS

Setup

Obtain the CSS file (and this HTML demo) from GitHub.

Reference the CSS, and font dependencies, in your HTML:

<head>
    <link rel="stylesheet" href="muse.css">
    <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Consolas|Montserrat">
</head>

Basic tags are pre-styled:

<body> <p> <a> <table> <tr> <th> <td> <code> <hr> <button> <input>
